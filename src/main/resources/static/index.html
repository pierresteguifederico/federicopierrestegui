My Projects

Here are a few of the projects I've previously worked on:

"PipiCucu Food"

It was my first project working as part of a team on an emulation project. Initially, it was meant to be a ten-person team with different roles, but in the end, we were just three people covering all the roles. As a first-year student, I had little experience, but I pushed forward, helping to build an MVP that we presented at "Demo Day" alongside 20+ teams. It was nerve-wracking, but I learned a lot about agile methodologies, teamwork, Spring Boot, MVC, and other concepts that the following year I encountered in subjects like System Analysis II and Web Applications. The project was about a web app for food delivery (for restaurant owners and customers).

"Sales Management"

Later on, I took an online course about SpringBoot to enhance and solidify my knowledge and concepts, and made a sales management web app. I developed a RESTful web page that implemented full CRUD operations (Create, Read, Update, Delete) using Spring Boot. This application allowed users to load, modify, and delete resources through RESTful APIs. Additionally, I created the front-end of the page using HTML, CSS, and JavaScript to consume these APIs, ensuring a seamless interaction between the client and the server.

"Library Management"

I also created a library management application, which allowed performing CRUD operations on books. This time, the project was a console-based application using SQLite as the database. I implemented design patterns such as Dependency Injection, Single Responsibility, and the DAO (Data Access Object) pattern to ensure scalability and maintainability. Additionally, I used JUnit to write unit tests and ensure the application's functionality and stability.
